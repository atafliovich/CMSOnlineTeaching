CMS online teaching refrence sheet

Livestreaming

  • BB colaborate
    • intergrated into quercus
    • record for later use, screensharing, breakout groups

  • Zoom (link)
    • many onlne tutorials, easy to use
    • record for later use, screensharing, breakout groups, remote access

    Ultimately Zoom has more fetures and is easier to learn to use/trouble shoot, while BB colaborate is better if the proffesor wants to keep everythung on quercus (and let students easily find recordings of past lectures on zoom)

Assignment Collection

  • Quercus Assignments
    • intergrated into quercus
    • can be easily used to get files from students
    • can add anotaions and comments

  • Markus
    • can be easily used to get files from students
    • can add anotaions and comments
    • show results of tests on collected software

  • Crowd Mark (link)
    • can be easily used to get individual pages from collected assignment
    • very easy to clearly show where student lost marks and add anotaiosn to student work

    Markus is good for collecting programming assignments, while crowd mark is good for collecting written work (such as math homework). As for quercus, it is already used by many students and as such would simplify their lives if it was used to collect assignments.

Editing Software

  • Openshot (link)
    • Free
    • Simple and easy for beginners to use
    • Can use to do all basic video editing (trim, animation, speed control, etc)
    • Not good for more complicated video editing

  • Davinci resolve (free version) (link)
    • visual effects, motion graphics, color correction and audio post-production
    • Useful for more complicated editing
    • Harder for beginners to use

  • Sony vegas pro (link)
    • Industry-leading software
    • Useful for making extremely well edited lecture
    • Costs money (depending to which version is acquired)
    • can be complicated and hard to learn

    For most proffessors Openshot will be the best choice since it has all the features needed for simple editing ad its easy to use for beginners. While Davinci resolve may be needed for professors who want to do more with their lectures but most (if not all) will not need any of its special features. Finally, unless a professor is trying to edit a complicated video sony vegas pro is outside of the requirements.
